File Extensions and their meaning:
All - rxe, rtm, rpg, rso, ric, rmd
Executeable - rxe, rtm
Program Files - rpg
Sound files - rso
Icon files - ric
Melody files - rmd
Datalog files - rdt
System files - sys
Menu files - rms
